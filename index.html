Silsilah — Family Tree

Genealogy & family tree builder

A cross-platform family tree application to build, visualize, and explore your genealogy. Released on iOS and Web, with the Android app launching soon.

Tech Stack Focus

  • SwiftUI: Custom relational canvas utilizing line-drawing paths (Bézier curves) for family connections with responsive pan/zoom gesture interaction.
  • SwiftData: Deep relational data graphs representing parents, children, and marriages, featuring custom cascading delete rules and schema integrity.
  • Kotlin: Native Android layer with Jetpack Compose, optimized for rendering complex tree layouts.
  • SvelteKit: High-efficiency web version designed with a custom grid system to ensure accessibility and SEO indexability of historical branches.

Neraca

Personal finance, simplified

An iOS personal finance app designed to track wallets, transactions, budgets, debts, and savings goals.

Tech Stack Focus

  • SwiftUI: Modular UI panels with dynamic widgets, gauge charts for budget utilization, and custom transition animations.
  • SwiftData: Highly optimized query descriptors to filter transactions and aggregate wallet balances in real-time.
  • Swift (Core): Built on decimal arithmetic architecture to prevent floating-point rounding discrepancies during multi-currency calculations.
